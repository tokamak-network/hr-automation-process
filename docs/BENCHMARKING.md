## 4. 기타 서비스

### PayHere (페이히어)
- **포지셔닝**: 소규모 사업자 급여 자동화
- **주요 기능**: 급여 계산, 4대보험, 원천징수, 급여명세서 자동 발송
- **장점**: 급여 특화, 소규모 최적, 저렴
- **단점**: HR 기능 거의 없음 (근태/채용/성과 없음), 크립토 미지원

### 니어HR (NIHR)
- **포지셔닝**: 중소기업 인사관리
- **주요 기능**: 인사정보, 근태, 급여, 전자결재
- **장점**: 심플, 저렴
- **단점**: 기능 제한적, 채용 없음, 크립토 미지원

### 하모니HR / 그리팅 (채용 특화)
- 그리팅: 채용 ATS 특화 (지원자 관리, 면접 스케줄링)
- 하모니HR: 인사·급여 올인원 (중견기업 타겟)

## Tokamak Network 관점 핵심 시사점

1. **크립토 급여를 지원하는 HR 솔루션은 국내에 없다** — USDT/토큰 기반 급여 관리는 자체 개발 필수
2. **독립계약자 모델 + 소득세 시뮬레이션**이라는 독특한 요구사항 → 기존 솔루션은 근로자 기반 설계
3. **채용 시스템이 이미 존재** (Tokamak Hiring System) — 이를 통합하는 것이 효율적
4. **10명 규모에서 flex 같은 풀스택은 과도** — 필요한 기능만 경량으로 구축하는 것이 합리적
5. **벤치마킹할 UI/UX**: flex의 대시보드 구조, 원티드스페이스의 Slack/Calendar 연동, 샤플의 모바일 우선 접근
6. **업비트 API 연동**으로 토큰 가격 자동 수집 → 기존 솔루션에 없는 차별점
