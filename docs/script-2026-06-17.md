## Slide 3 — Tax Calendar (50 sec)

Singapore has many tax deadlines — GST every quarter, ECI, IR8A, Form C filing.

We built a Tax Calendar page. It shows all deadlines in a calendar view or list view. Each deadline has a category, YA year, and D-Day countdown. You can filter by category or status.

Overdue items show in red. You can mark items as completed when done.

We also synced all 12 deadlines to Google Calendar — the Tokamak HR calendar. Each event has D-7 and D-1 popup reminders. So you get notified before any deadline.

## Slide 4 — Expense Approval (60 sec)

Before, expenses were just a list. No approval process, no currency conversion.

Now we have a full flow. When someone submits an expense — for example, Member 3's Anthropic subscription in EUR — the system does three things:

One, it maps the submitter name to a team member using Drive folder names.

Two, it converts the original currency to USDT using ECB exchange rates. EUR 216 becomes about 250 USDT.

Three, it goes to the approval screen as "pending."

The operator can then click Pay, Hold, or More Docs. When you click Pay and enter a payment date, the system gets the D-1 exchange rate and records the confirmed USDT amount. No actual money is sent — it only records the decision.

When approved, it immediately shows in the expense list — same data, no manual sync needed.
