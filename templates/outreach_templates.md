# Outreach Templates

## 1. LinkedIn DM — 첫 접촉 (Cold Outreach)

### English
```
Hi {name},

I came across your work on {project/skill} and was really impressed. I'm with Tokamak Network, an Ethereum L2 protocol team.

We're looking for talented developers who can contribute to our ecosystem — and we do hiring differently. No resumes, no leetcode puzzles. Instead, we evaluate candidates based on actual deliverables (we call it "Track B").

You'd build something for the Tokamak ecosystem, we'd review your work, and if it's a fit, we move fast.

Would you be open to learning more?

Best,
{sender_name}
Tokamak Network
```

### 한국어
```
안녕하세요 {name}님,

{project/skill} 관련 작업을 보고 연락드립니다. 저는 이더리움 L2 프로토콜 Tokamak Network 팀입니다.

저희는 채용 방식이 좀 다릅니다. 이력서나 알고리즘 테스트 없이, 실제 결과물(Track B)로 평가합니다. 토카막 생태계에 기여할 수 있는 프로젝트를 만들어 제출하면, 팀이 직접 리뷰합니다.

관심 있으시면 자세한 내용 안내드리겠습니다.

감사합니다,
{sender_name}
Tokamak Network
```

## 3. Email — 공식 채용 제안

### Subject: Tokamak Network — Build & Join (No Resume Needed)

### English
```
Dear {name},

I found your profile through {source: GitHub/LinkedIn} and your experience in {skill_area} caught our attention.

About Tokamak Network:
We're an Ethereum Layer 2 protocol focused on on-demand, tailored L2 solutions. Our team of 15 engineers manages 70+ active repositories with 2,000+ commits every two weeks.

Our Hiring Process (Track B):
1. Pick a challenge from our ecosystem (or propose your own)
2. Build a working deliverable (code, demo, or docs)
3. Submit for team review
4. If approved: demo session + fast-track offer

What we offer:
• $50-100 interview compensation for Track B submission
• Remote-first, global team
• Direct contribution to Ethereum L2 infrastructure
• Competitive compensation in crypto + fiat

No resumes. No algorithm puzzles. Just real work.

Interested? Reply to this email or check out: {link}

Best regards,
{sender_name}
{sender_title}
Tokamak Network
https://tokamak.network
```

### 한국어
```
{name}님께,

{source}에서 프로필을 확인하고 {skill_area} 경험이 인상적이어서 연락드립니다.

Tokamak Network 소개:
이더리움 Layer 2 프로토콜로, 맞춤형 L2 솔루션을 제공합니다. 15명의 엔지니어가 70개 이상의 활성 레포지토리를 관리하며, 2주마다 2,000건 이상의 커밋을 기록하고 있습니다.

채용 프로세스 (Track B):
1. 생태계 챌린지 선택 (또는 직접 제안)
2. 작동하는 결과물 빌드 (코드/데모/문서)
3. 팀 리뷰 제출
4. 승인 시: 데모 세션 + 빠른 오퍼

제공 사항:
• Track B 제출 시 면접비 $50-100 지급
• 리모트 우선, 글로벌 팀
• 이더리움 L2 인프라에 직접 기여
• 크립토 + 법정화폐 혼합 보상

이력서 없음. 알고리즘 테스트 없음. 오직 실제 결과물.

관심 있으시면 회신 부탁드립니다: {link}

감사합니다,
{sender_name}
{sender_title}
Tokamak Network
```

## Variables Reference

| Variable | Description | Source |
|----------|-------------|--------|
| `{name}` | Candidate full name | LinkedIn/GitHub profile |
| `{project/skill}` | Relevant project or skill | Profile analysis |
| `{skill_area}` | Technical expertise area | AI analysis |
| `{source}` | Where we found them | github/linkedin |
| `{repo_name}` | Tokamak repo they interacted with | Monitor data |
| `{action}` | starred/forked/contributed to | Monitor data |
| `{sender_name}` | Recruiter name | Current user |
| `{sender_title}` | Recruiter title | User profile |
| `{link}` | Track B info page URL | TBD |
| `{link_to_track_b_info}` | Track B detailed guide | TBD |
